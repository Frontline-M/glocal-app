# Glocal 1.0.3+4

Release date: 2026-03-13
Git tag: v1.0.3-4

## Highlights
- Added a shared spoken localisation layer for all declared app languages.
- Localised spoken time, weather, calendar, reminder, and missed-reminder phrases through one catalog.
- Wired weather descriptions into the same spoken phrase system to avoid partial English output.
- Preserved scheduler, storage, notification timing, background runtime, and TTS engine selection logic.
- Included the earlier timezone, exact-alarm, reminder cleanup, and TTS locale resolver fixes in the pushed repo state.

- flutter build appbundle --release
