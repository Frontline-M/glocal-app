# Known Limitations and Phase 2 Roadmap

## Current limitations
- Desktop screensaver mode is scaffolded as a phase 2 target.
- On-device STT availability depends on installed language packs.
- Hourly background behavior varies by OEM battery optimization policies.
- TTS language quality depends on platform voice pack support.

## Phase 2 roadmap
- Windows/macOS screensaver shell integration.
- Secure local encryption layer for reminder content at rest.
- Cloud STT fallback provider with explicit opt-in.
- Rich reminder parser (`tomorrow 8am`, recurring schedules).
- Multi-location weather profiles.
