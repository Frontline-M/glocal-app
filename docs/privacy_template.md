# Privacy Policy Template (GDPR/CCPA-ready starter)

Glocal processes only the minimum data needed for clock, weather, and reminders.

## Data used
- Location: used to fetch local weather.
- Microphone audio: used only for reminder transcription.
- Reminder content: stored locally on device.
- App settings: language, voice, volume, quiet hours.

## Data storage
- Settings and reminder metadata are stored locally.
- No sale of personal data.

## User controls
- Permissions can be revoked in system settings.
- Reminders can be deleted in app.
- Users can disable weather and voice features.
